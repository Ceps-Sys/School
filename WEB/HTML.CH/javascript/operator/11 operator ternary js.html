Operator Ternary

Operator ternary merupakan operator yang terdiri dari tiga bagian,pada JavaScript operator tenary biasa digunakan untuk membuat sebuah percabangan/kondisi if/else :

  1. kondisi (condition)
  2. nilai jika kondisi nenar (expression if true)
  3. nilai jika kondisi salah (expression if false)
        condition? expressionIftrue : expressionIfFalse
