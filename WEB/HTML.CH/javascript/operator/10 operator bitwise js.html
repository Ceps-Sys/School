operator bitwise merupakan operator yanhg digunakandi operasi bit (binary) pada nilai-nilai bilangan bulat. operator bitwise terdiri atas:

  • Bitwise AND (&)
  • Bitwise OR (|)
  • Bitwise XOR (^)
  • Bitwise NOT (~)
  • Bitwise Left Shift (<<)
  • Bitwise Right Shift (>>)
  • Bitwise Unsigned Right Shift (>>>)

Operator ini merupakan binner jadi jika ada variabel di isi dengan desimal maka dia akan di olah menjadi binner

Operartor ini berlaku unntuk tipe data int,long,short,char dan byte
