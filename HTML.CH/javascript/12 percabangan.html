Apa itu percabangan dan kenapa di namakan percabangan?

Percabangan adalah sebuah konsep dalam pemrograman yang mempermudah seseorang untuk membuat keputusan berdasarkan kondisi tertentu.Dinamakan bercabang karena program dapat "Bercabang" ke jalur yang berbeda tergantung pada hasil evaluasi kondisi tersebut.

Dalam JavaScript percabangan biasanya dilakukan melalui struktur kontrol seperti If,Else If, dan Else.Dalam JavaScript terdapat 6 bentuk percabangan:

  1. If
  2. If...Else
  3. If...Else If...Else
  4. Switch
  5. Ternary Operator
  6. Short-circuit Evaluation
