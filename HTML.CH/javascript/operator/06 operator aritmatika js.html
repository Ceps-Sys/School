Apa itu Operator?

Operator adalah simbol yang digunakan untuk melakukan operasi pada suatu nilai atau variabel.

Operator dalam pemrograman terbagi menjadi 6 jenis:

  1. Operator Aritmatika;
  2. Operator penugasan (Assigment);
  3. Operator Relasi;
  4. Operator Logika;
  5. Operator Bitwise;
  6. Operator Ternary;
